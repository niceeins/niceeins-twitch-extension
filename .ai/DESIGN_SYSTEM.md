# NiceEins Design System

## 1. Design-Richtung
NiceEins ist ein dark-first Creator-SaaS für Twitch-Streamer. Die UI soll modern, ruhig, gaming-nah und professionell wirken.

## 2. Prinzipien
- Clarity over decoration
- Mobile-first
- Cards over tables
- One primary action per section
- Accessibility by default
- Calm motion only
- Consistent spacing
- No random one-off styles

## 3. Visual Style
- Dark-first
- Bento/Card Layouts
- Dezente Glows nur für Status oder Highlights
- Akzentfarbe gezielt einsetzen
- Keine grellen Vollflächen
- Keine Textüberlagerung auf unruhigen Bannern ohne Lesbarkeitsschicht

## 4. Tokens
- Spacing Scale
- Radius Scale
- Typography Scale
- Shadow Levels
- Color Roles
- Status Colors

## 5. Components
- Buttons
- Cards
- Badges
- Forms
- Tabs
- Toasts
- Modals
- Empty States
- Skeletons
- Schedule Cards
- Game Cards
- Command Rows

## 6. Page Patterns
- Dashboard Overview
- Stream Management
- Games Management
- Profile Settings
- Public Profile
- Twitch Panel

## 7. Accessibility
- Keyboard usable
- Visible focus
- Sufficient contrast
- Reduced motion support
- Labels for forms
- No color-only meaning

## 8. AI Implementation Rules
Before changing UI, inspect existing components.
Reuse existing classes where possible.
Do not create isolated styles unless necessary.
After changes, check mobile, spacing, contrast, focus states and empty states.
